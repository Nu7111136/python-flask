# Git SSH Troubleshooting Guide

## 1️⃣ Setting Up SSH Key
### Generate a New SSH Key
```sh
ssh-keygen -t rsa -b 4096 -C "your-email@example.com"

#### 2 Saving the SSH key
Save it in ~/.ssh/id_rsa
Copy the public key:
cat ~/.ssh/id_rsa.pub

### 3 Setting it on the github
Add it to GitHub: Settings > SSH and GPG keys > New SSH Key

### 4 Test SSH Connection
 ssh -T git@github.com 

### 5 Check Remote URL
git remote -v

### 6 If it's using HTTPS, change it to SSH (need based)
git remote set-url origin git@github.com:<username>/<repo>.git


/////////////////////

DEBUG (IF SSH is not working)

ssh-add -l
ssh-add ~/.ssh/id_rsa (add manually if not  listed)
ssh -T git@github.com (check again)

Git Commands to Check Status
git ls-remote origin



Quick Fix Summary
Issue	Fix
No SSH key found			ssh-keygen -t rsa -b 4096
Permission denied			Add key using ssh-add ~/.ssh/id_rsa
Git using HTTPS instead of SSH		git remote set-url origin git@github.com:<username>/<repo>.git
Cannot pull/push			git fetch origin && git pull origin main
Debug SSH				GIT_SSH_COMMAND="ssh -vvv" git pull origin main
